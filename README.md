# ComiCool

An online bookstore specializing on Japanese comic books! You can now browse and buy any manga, any volume you want!

## Technologies Used
- Frontend: HTML, CSS, React, Javascript
- Backend: Node.js, Express, Javascript
- Database: MongoDB
- Authentication: JWT
- Deployment: Heroku
